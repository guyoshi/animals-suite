# Guia rápido — versão 0.9.0

## Terra de Gaia

1. Abra **Terra de Gaia**.
2. Procure a área na biblioteca esquerda.
3. Arraste-a para o canvas.
4. Segure um conector amarelo do primeiro nó.
5. Arraste até o segundo nó e solte sobre ele.
6. Para apagar, clique na linha e pressione `Delete`.
7. Para recomeçar pela topologia oficial, clique em **Resetar mapa original**.

O reset preserva imagens e desenhos, mas substitui nós e ligações.

## Voltar para a página anterior

- Use a seta no canto superior esquerdo para voltar exatamente à última página.
- Clique no nome do mundo ou da área no caminho superior para voltar diretamente a esse nível.

## Água circular

1. Abra o mapa de uma área.
2. Selecione a ferramenta **Água**.
3. Escolha **Circular**.
4. Desenhe a zona no canvas.

## Importar mecânicas criadas por IA

1. Abra **Mecânicas**.
2. Clique em **Exportar base para IA**.
3. Envie o JSON para a IA e solicite novas mecânicas no mesmo formato.
4. Clique em **Importar mecânicas**.
5. Selecione o JSON recebido.
6. Em cada duplicado, compare:
   - nova à esquerda;
   - atual à direita.
7. A opção padrão mantém a mecânica atual.
8. Confirme a importação.

## Ajuda

- Abra **Ajuda** no menu lateral.
- Use os botões `?` para abrir a explicação do campo ou sistema atual.
- **O que há de novo** fica no pequeno botão da barra superior.

## Música

As fichas já foram criadas. Abra **Músicas**, escolha uma ficha e adicione o arquivo de áudio.

Os controles **Repetir** e **Fixar** ficam na barra musical superior.
