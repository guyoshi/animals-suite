# Animals Suite 1.0.1

## Correção — músicas anexadas ao Planejador

- O player do Animals — Executor passa a considerar exclusivamente os arquivos de áudio anexados às fichas de Música no Planejador.
- Referências sem anexo reconhecido não aparecem na playlist do Executor.
- Ao anexar um áudio, o Planejador registra nome original, tipo, tamanho e data do anexo.
- Arquivos já adicionados em versões anteriores são migrados como anexos do Planejador para preservar o projeto existente.
- A interface agora distingue claramente “áudio anexado”, “sem áudio anexado” e “anexo não encontrado”.
- O player musical do próprio Planejador também usa apenas anexos reconhecidos.
- A validação das músicas das vilas só considera completa uma faixa que possua áudio anexado.

Nenhuma ficha de música, relação, anotação ou configuração existente é apagada.
